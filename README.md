# How to Start the Server
1. Start docker and remove any running containers
2. Run postgresql remote image in docker using terminal: 
```
$ Docker run --name postgres -p 5432:5432 -e POSTGRES_PASSWORD=postgres -d postgres
```
3. Create a postgres db collection with password & name = postgres in dbeaver 
4. Create a db called "olliodb" in postgres 


## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ yarn install
```

## Running the client

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Running the db on docker
```bash
# list all containers
$ docker ps -a

# stop container
$ docker stop "container id"

# remove container
$ docker rm "container id"

# run postgresql remote image in docker
$ docker run --name postgres -p 5432:5432 -e POSTGRES_PASSWORD=postgres -d postgres

# run postgresql server image in docker
$ docker compose up -d db
$ docker compose build
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
